## What is inside?

This project uses lot of stuff as:

- [EJS](https://ejs.co/)
- [Google Sheets](https://developers.google.com/sheets/api)
- [Send Grid](https://sendgrid.com/)

## Getting Started

```bash

npm start

```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
